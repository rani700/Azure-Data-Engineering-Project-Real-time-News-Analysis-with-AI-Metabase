This project implements a serverless, end-to-end Azure Data Engineering pipeline designed to ingest, enrich, and visualize real-time news data about "Electric Vehicles." Orchestrated by Azure Data Factory (ADF) on a daily schedule, the workflow triggers a Python-based Azure Function that interfaces with an external conversational AI agent API (news-agent.codeshare.live) to extract raw articles, employing custom regex logic to normalize relative timestamps (e.g., "6h ago") into standard ISO 8601 format. Once the data is structured, ADF executes a ForEach loop to pass individual articles to Azure AI Language services via Web Activities, enriching the content with sentiment scores, entity extraction, and key phrases. The final enriched dataset is upserted into an Azure SQL Database using stored procedures to ensure schema consistency, while Metabase is connected as the downstream BI layer for dashboarding.
